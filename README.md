🌍 TravelNest
A full-stack travel booking and exploration platform built using the MERN (MongoDB, Express.js, React, Node.js) stack. This application allows users to discover exotic destinations, view detailed itineraries, and manage their travel bookings seamlessly.

🚀 Key Features
User Authentication: Secure login and signup using JWT (JSON Web Tokens).

Destination Discovery: Browse through various travel packages with high-quality images and descriptions.

Booking System: Integrated form for users to book tours and select travel dates.

Admin Dashboard: A dedicated panel for administrators to add, update, or delete travel packages (CRUD operations).

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS / Bootstrap

Backend: Node.js, Express.js

Database: MongoDB (with Mongoose)

State Management: Context API / Redux
